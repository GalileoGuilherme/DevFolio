Desenvolvimento WEB

REACT

O React é uma biblioteca JavaScript de código aberto com foco em criar interfaces de usuário em páginas web. É mantido pelo Facebook, Instagram, outras empresas e uma comunidade de desenvolvedores individuais. É utilizado nos sites da Netflix, Imgur, Feedly, Airbnb, SeatGeek, HelloSign, Walmart e outros.

Web Development

Desenvolvimento web é o termo utilizado para descrever o desenvolvimento de sites, na Internet ou numa intranet. O profissional que trabalha desenvolvendo websites pode ser um web designer ou um web developer.



Bootstrap

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo.

Responsive Design

O design responsivo é uma técnica que permite que a interface de um site ou aplicativo seja adaptada em qualquer tipo e tamanho de tela, tornando-a responsiva, ou seja, sem distorções ou inacessibilidade no conteúdo em todos os dispositivos.

Ux

UX é a abreviação de User Experience, em português, experiência do usuário. Trata-se de uma estratégia que tem como objetivo entregar experiências melhores aos usuários de serviços e produtos digitais ou físicos.

Quality Assurance

O Quality Assurance é o conjunto de atividades que tentam garantir que o produto ou serviço oferecido estejam de acordo com o nível de qualidade exigido.

Reciclagem

Como descartar seu notebook usado – e ganhar desconto em um novo?

A importância da reciclagem é um assunto comum atualmente, mas ainda há muitas questões a serem consideradas quando falamos dessa problemática da sociedade. A reciclagem de materiais eletrônicos, por exemplo, não é tão difundida quanto pensamos, mas merece nossa atenção por tratar de itens que, quando não são descartados corretamente, podem gerar consequências severas para a comunidade e a natureza. Existem empresas engajadas na conscientização do público, como a Intel, que por meio do Recicla PC Think Circular vem promovendo uma série de ações para divulgar a importância da reciclagem de eletrônicos. Para dar um incentivo extra a essa prática, a companhia está oferecendo descontos surpreendentes a quem descartar corretamente um computador usado e desejar investir em um novo.
